# dataset-DZ2
В файле датасет с продажами остросезонного зимнего товара стеклоомыватель, в г.Минске с учетом влияния погодных условий, с дальнейшим прогнозированием продаж с учетом прогноза погода на блишайших 10 дней. В ходе работы создается матрица о продажах в разрезе календарных дней и дней нелели, с учетом показаний дневной и ночной температуры, а также погодные условия (снег, дождь, солнце, пасмурно), и строится визуализация.
Констанстная сложность проекта (0(1): алгоритм выполняется за постоянное количество временни, независимо от размера данных

Датасет содержит следующие столбцы:

Date - календарная дата 

dayweek - день неледи (1-7)

sales - кол-во продаж в шт.

tempday - дневная температура зафиксированная в 15.00 ежедневно

tempnight - ночная температура зафиксированная в 03.00 ежедневно

precipitation - погодные условия (снег, дождь, солнце, пасмурно)
