# dataset-DZ2
В файле датасет с продажами остросезонного зимнего товара стеклоомыватель, в г.Минске с учетом влияния погодных условий, с дальнейшим прогнозированием продаж с учетом прогноза погода на блишайших 10 дней.
Необходимо создать матрицу о продажах в разрезе календарных дней и дней нелели, также показания дневной температуры и ночной, а также погодные условия (снег, дождь, солнце, пасмурно).
Констанстная сложность проекта (0(1): алгоритм выполняется за постоянное количество временни, независимо от размера данных
